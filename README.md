# Linear TG Bot

Telegram-бот для управления задачами в Linear. Создаёт задачи голосом, текстом или форвардом — прямо из Telegram.

## Возможности

- **Текст → задача** — любое сообщение создаёт issue в Linear с автоопределением приоритета
- **Голосовое → задачи** — расшифровка через Groq Whisper + извлечение задач через LLaMA
- **Форвард → задача** — перешли любое сообщение, бот разберёт и создаст задачу
- **Reply → комментарий** — ответь на сообщение бота, комментарий улетит в Linear
- **`/todo`** — список активных задач с кнопками удаления
- **`/help`** — все команды
- **Онбординг** — кнопка с материалами для новых участников команды

## Стек

- [grammY](https://grammy.dev/) — Telegram Bot framework
- [Linear GraphQL API](https://developers.linear.app/) — создание и управление задачами
- [Groq](https://groq.com/) — транскрипция голоса (Whisper-large-v3) + извлечение задач (LLaMA 3.3 70B)
- [Railway](https://railway.app/) — деплой и хостинг

## Установка

```bash
git clone https://github.com/Pupeca/linear-tg-bot
cd linear-tg-bot
npm install
```

Создай `.env`:

```env
TELEGRAM_BOT_TOKEN=...
LINEAR_API_KEY=...
LINEAR_TEAM_ID=...
GROQ_API_KEY=...
```

```bash
npm start
```

## Команды

| Команда | Описание |
|---|---|
| `/start` | Запустить бота |
| `/todo` | Список активных задач |
| `/help` | Все команды и подсказки |

## Приоритеты

Бот определяет приоритет автоматически по ключевым словам:

| Слова | Приоритет |
|---|---|
| срочно, asap, горит | 🔴 Срочно |
| высокий, важно, нужно | 🟠 Высокий |
| средний, medium | 🟡 Средний |
| низкий, low | 🟢 Низкий |

Или явно: `высокий - доделать документацию`

## Деплой на Railway

1. Fork или push на GitHub
2. [railway.app](https://railway.app) → New Project → Deploy from GitHub
3. Variables → добавить все 4 ключа из `.env`
